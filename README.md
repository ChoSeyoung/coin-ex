## 셋업 | Project setup

```bash
# 아래 명령어 실행 후 .env.local 에 ACCESS, SECRET KEY 작성
$ cp .env.production .env.local

# 의존성 패키지 설치
$ npm install
```

## 프로젝트 실행 | Project run

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
