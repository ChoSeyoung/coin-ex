## 셋업 | Project setup
env 에 텔레그램 BotFather 봇 관련 BOT_TOKEN, CHAT_ID 를 입력하면 알림을 받을 수 있습니다. 
```bash
# 아래 명령어 실행 후 .env.local 에 ACCESS, SECRET KEY 작성
$ cp .env.production .env.local

# 의존성 패키지 설치
$ npm install
```

## 프로젝트 실행 | Project run

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
